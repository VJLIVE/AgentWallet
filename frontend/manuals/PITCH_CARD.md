## 🎯 One-Line Pitch
**"We built a system where AI agents can spend money autonomously — but never overspend."**

---

## 🔥 The Problem (30 seconds)
- AI agents are becoming autonomous
- They can call APIs, execute workflows, make decisions
- **But they cannot safely handle money**
- No control layer exists for AI spending
- Existing systems assume humans initiate payments

---

## 💡 The Solution (30 seconds)
**AgentWallet** = Autonomous Payment Guardian

**How it works:**
1. AI agent attempts payment
2. System validates against blockchain rules
3. Returns structured decision (approved/blocked/modified)
4. Agent adapts and retries if needed
5. Complete transparency with audit logs

---

## 🎬 The Demo (3 minutes)

### Scenario: "Summarize PDF under 0.5 ALGO"

**What happens:**
1. ✅ Agent plans task → needs PDF parser + summarizer
2. ❌ Tries expensive service (2 ALGO) → BLOCKED
3. 🔄 Agent optimizes → switches to cheap service (0.5 ALGO)
4. ✅ Payment approved → Task completes

**The Magic:** Agent autonomously adapts to spending constraints
